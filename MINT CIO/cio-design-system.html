✓ Do
  • Always use currentColor — never hardcode fills
  • Use --icon-md (20px) as the default size
  • Add aria-label to icon-only buttons
  • Use outline style; filled for selected/active states
  • Match icon weight to surrounding typography
✗ Don't
  • Don't mix icon libraries on the same surface
  • Don't use non-tokenised sizes (e.g. 18px, 22px)
  • Don't use icons as sole communication of status
  • Don't use decorative icons without aria-hidden="true"
  • Don't hardcode icon colors — use semantic tokens
