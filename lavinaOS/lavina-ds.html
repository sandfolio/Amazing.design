03 — Components

Button components

There's no filled/solid button anywhere on the site — every actionable element is an outline, a bare link, or a hairline chip that inverts to ink on hover. Hover the stage to see each state live.

Primary — Ghost button

The only true "button." Used once per project card to open a case study.

Default transparent fill · 1px ink border · ink text
Hover inverts to ink fill · paper text · arrow shifts +3,-3
Transition background/color/transform .25s ease
